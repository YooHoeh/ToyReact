# DIY a simple React Frame;

## Day 1

```shell
npm init -y

npm i webpack-cli -g # 只执行一次，用于运行webpack打包命令

npm i webpack --save-dev
npm i @babel/core --save-dev    # babel核心，用于代码转换
npm i @babel/plugin-transform-react-jsx --save-dev  # 用于代码转换jsx形式代码
npm i @babel/preset-env --save-dev
npm i babel-loader --save-dev
npm i webpack-dev-server --save-dev
```
